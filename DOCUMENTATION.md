# JobSphere Documentation

## Project Overview
JobSphere is a responsive job board application built using Next.js.

## Features
- Browse jobs
- Search jobs
- View job details
- Apply for jobs
- About page
- Responsive design

## Architecture
- Components
- API Route
- Data Layer
- Dynamic Routes

## CI/CD
- GitHub Actions validates every push.
- Vercel automatically deploys the latest version.

## Future Improvements
- User authentication
- Database integration
- Real job API
- Resume upload
- Saved jobs
