# JobSphere

JobSphere is a modern, responsive job board web application built with **Next.js 15**, **TypeScript**, and **Tailwind CSS**. The project was developed as a Software Engineer assessment to demonstrate modern frontend development, routing, API integration, responsive design, and CI/CD deployment.

---

## Features

- Responsive and modern user interface
- Browse featured job listings
- Search jobs by title
- Filter jobs by location
- Dynamic job details page
- Job application form
- About page
- Breadcrumb navigation
- Loading page
- Custom 404 page
- REST API using Next.js Route Handlers
- GitHub Actions CI pipeline
- Automatic deployment with Vercel

## Installation

Clone the repository:

```bash
git clone https://github.com/Arun1414-tech/jobsphere.git
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```
http://localhost:3000
```

---

## CI/CD

This project uses GitHub Actions for Continuous Integration.

Every push to the **master** branch automatically:

- Installs dependencies
- Builds the project
- Verifies successful compilation

The application is deployed automatically through **Vercel**, providing Continuous Deployment.
